# 网络流量分析与异常检测系统

这是一款基于图拓扑模型构建的高性能网络流量分析与威胁审计系统。系统采用 `C++` 作为底层计算引擎，配合 `Python` 现代图形化界面与交互式图渲染技术，实现了从原始报文解析、会话流聚合、拓扑特征提取到多维风险画像的全链路安全审计闭环。

---

## 🚀 核心功能 (Core Features)

### 1. 深度流量审计与流重组 (Traffic Auditing)
* **高效图拓扑构建**：底层采用优化后的邻接表（Adjacency List）架构，实时聚合网络五元组会话流，精准统计单边通信的端口数量、时长及负载大小。
* **协议级流量排序**：支持对全网节点总吞吐量进行全局排序，并内置针对 `HTTPS (TCP:443)` 协议的专项隐蔽流量排行榜。
* **全网广撒网扫描检测**：基于图出度异常与“发出流量占比 > 80%”特征，快速定位全网盲扫僵尸网络。
* **智能路径规划**：
  * **最小跳数路径**：通过 BFS 算法快速计算节点间的逻辑最短连通链路。
  * **最小拥塞路径**：基于动态权重的 Dijkstra 算法，以“流量/时长”为代价矩阵寻找最优负载路径。

### 2. 多维威胁检测与资产画像 (Threat Detection & Profiling)
* **精准扫描器捕获**：
  * **单点爆破扫描检测**：突破传统单向流量检测盲区，精准识别对单一目标 IP 发起高频端口探测（单边关联会话数 $\ge$ 15 且均载极小）的针对性渗透源。
* **资产角色自动画像**：根据节点的出入度模型与业务吞吐量，自动推断网络节点角色（如：`[*] 核心服务器`、`[+] 核心枢纽节点`、`[!] 疑似扫描器` 及普通终端）。
* **星型拓扑探测**：智能识别连接数达标（$\ge$ 20）且符合星型辐射特征的核心网关或 C2 枢纽节点，支持容忍阈值 $k$ 灵活配置。

### 3. 安全风控与交互式分析 (Risk Control & Analysis)
* **量化风险评估矩阵**：综合“扫描行为确信度”、“数据泄露风险”、“拓扑核心影响面”及“异常流量大户”四大维度，为每个节点计算 `0-100` 的标准化安全风险评分。
* **交互式图透视 (Visualizer)**：利用 `Pyvis` 物理引擎，一键生成指定目标节点的连通子图。节点颜色直观映射安全风险评级（🔴高危 / 🟡警告 / 🟢正常），支持动态拖拽、缩放及详细悬停画像分析。
* **动态安全规则引擎**：支持自定义目标 IP 与 IP 区间白名单/黑名单的细粒度通信阻断审计，实时发现违规越权通信行为。

### 4. 自动化数据管道 (Data Pipeline)
* **PCAP 离线提取器**：内置基于 `Scapy` 的数据转换引擎，支持直接读取通用 `.pcap` 原始抓包文件，自动清洗、过滤并聚合为系统直接可用的结构化 CSV 会话数据集。

## 🛠️ 技术栈选型 (Tech Stack)

| 领域 | 选型组件 | 核心作用描述 |
| :--- | :--- | :--- |
| **底层核心计算** | **C++ 17** | 负责高并发图模型构建、复杂路径搜索与风险矩阵量化运算 |
| **交互控制台** | **CustomTkinter** | 提供支持深色模式的现代化、非阻塞 GUI 指令调度控制台 |
| **图论与算法** | **Dijkstra / BFS / DSU** | 解决链路权重寻优、最短路由分析及连通分量子图提取问题 |
| **可视化引擎** | **Pyvis / NetworkX** | 基于力导向模型渲染交互式 HTML 节点拓扑图 |
| **数据采集与清洗**| **Scapy / Pandas** | 实现底层物理网卡抓包文件的报文解包与聚合降维分析 |
